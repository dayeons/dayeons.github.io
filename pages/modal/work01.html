티오더 CDP 플랫폼

  • (내부 관리 시스템 개발)
    고객 확보에서 유지까지 전 과정을 지원하는 고객 데이터 솔루션으로서 고객 데이터 관련 기능과 고급 툴을 제공
  • 기간: 2023년 3월 ~ 2024년 6월
  • 역할: 웹퍼블리싱 및 프론트엔드 지원
  • 담당업무
    • - 퍼블리싱 UI 작업
    • - 웹 표준 작업
    • - vue 프레임워크를 사용
    • - Ant design 활용한 table, tooltip, select, input 등 프론트 개발
    • - 다수의 프론트 개발을 공통 작업으로 정리
  • Slack, Jira, Confluence를 통해 팀과의 협업을 원활히 하고, 프로젝트 진행 상황을 주기적으로 공유하며 이슈를 해결
  • 매주 배포 완료 후, 사용자 만족도 조사를 통해 영업부, 영업지원팀 등 만족도 향상을 달성
  • 기술 스택
    • - html, scss, vue, ant vue, git, github, jira, confluence, notion, figma
